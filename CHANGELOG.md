## [0.2.0] — 2026-06-18

- `CODE_OF_CONDUCT.md` — contributor covenant adapted from v2.1
- `SECURITY.md` — vulnerability reporting policy with private disclosure process
- `.github/ISSUE_TEMPLATE/bug_report.md` — structured bug report template
- `.github/ISSUE_TEMPLATE/feature_request.md` — structured feature request template
- `.github/PULL_REQUEST_TEMPLATE.md` — pull request template with summary, testing, and checklist fields
- README links to all new community files
